# 单元测试目录

## 说明
本目录包含MAZE网络各个核心模块的单元测试，用于验证模块的独立功能正确性。

## 测试内容

### router_unit/
- **测试目的**: 验证故障感知XY路由算法
- **测试用例**:
  - 正常路由功能测试
  - 故障位置感知测试
  - 边缘节点处理测试
  - 多播/广播路由测试
  - 边界条件测试

### arbiter/
- **测试目的**: 验证QoS仲裁器功能
- **测试用例**:
  - QoS优先级测试
  - 多输入仲裁测试
  - 饥饿避免测试
  - 性能特性测试

### node/
- **测试目的**: 验证完整节点功能
- **测试用例**:
  - 5输入5输出功能测试
  - 并行路由处理测试
  - 故障容错测试
  - 性能基准测试

### topo/
- **测试目的**: 验证拓扑连接功能
- **测试用例**:
  - 节点间连接测试
  - 环绕连接测试
  - 缓冲器深度测试
  - 拓扑完整性测试

## 运行方法
```bash
# 运行所有单元测试
cd verification/testbench/unit_tests
make all

# 运行特定模块测试
cd router_unit
make test
```
