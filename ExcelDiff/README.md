# Excel表格比较工具

这是一个Python工具，用于比较两个Excel表格，找出指定列中内容不一致的所有行。

## 功能特点

- 比较两个Excel表格中的B、E、F列（可配置）
- 自动检测并忽略带有删除线格式的行（只要A-Z列中任意一列有删除线，整行都被忽略）
- 输出两个结果表格：表1独有的行和表2独有的行
- 支持自定义唯一标识列
- 自动生成测试数据

## 使用方法

### 1. 安装依赖

```bash
pip install pandas openpyxl
```

### 2. 运行程序

```bash
python excel_compare_tool.py
```

首次运行时会自动创建样例Excel文件进行测试。

### 3. 自定义配置

在 `excel_compare_tool.py` 文件的 `__init__` 方法中，您可以修改以下参数：

- `file1_path` 和 `file2_path`：输入Excel文件路径
- `unique_column`：作为唯一标识的列（默认"A"列）
- `compare_columns`：需要比较的列列表（默认["B", "E", "F"]）
- 输出文件名称

## 输出文件

- `only_in_table1.xlsx`：包含表1中有而表2中没有的行
- `only_in_table2.xlsx`：包含表2中有而表1中没有的行

## 示例

程序自动生成的样例数据包含以下类型的差异：
- 表1独有的行（ID005）
- 表2独有的行（ID006、ID007，但ID007被标记为删除线，所以只保留ID006）
- 内容差异的行（ID001、ID003、ID004）

## 注意事项

- 确保Excel文件格式正确
- 删除线格式：只要A-Z列中任意一列有删除线，整行都会被忽略
- 比较基于字符串匹配，区分大小写
- 空值会被转换为空字符串进行比较
